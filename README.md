# 📱 FAST NUCES Student Social Network

A dedicated social networking web and mobile application tailored specifically for students at FAST NUCES Lahore to communicate, share updates, and connect. This project was developed as part of our 4th-semester curriculum to replicate the core social features of platforms like Instagram.

## ✨ Key Features
- Cross-Platform Access: Works smoothly as both a web platform and a mobile application.
- Student Communication: Built directly for campus networking and peer interaction.
- Core Social Tools: Includes streamlined, Instagram-inspired features for sharing and user connection.
- Modern Codebase: Fully developed using TypeScript for reliable and structured application logic.

## 🛠️ Tech Stack
- Language: TypeScript (78.1%)
- Platform Target: Web Browsers and Mobile Devices

## 🎓 Course Context
- University: FAST NUCES Lahore
- Timeline: 4th Semester Group Project
